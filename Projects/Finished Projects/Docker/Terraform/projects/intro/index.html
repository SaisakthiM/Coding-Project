Technology Stack

🐍
Python / Django
Notes, Blog, Hospital, Document, Social backends. REST Framework + template views.
☕
Java / Spring Boot
Bank Manager API + Social Media Java microservice. JPA + Spring MVC.
🐹
Go Microservice
Social Media caching and real-time layer integrated with Redis.
🟩
Node.js / Express
Video Uploader and API Service backends with REST endpoints.
⚛️
React + Vite
All 7 frontend SPAs. TypeScript/JSX, fast HMR builds.
🌐
Nginx Gateway
Single entry point on :80. Proxies all 9 apps + serves static builds.
☸️
Kubernetes (kind)
Local K8s cluster for Social Media App. Helm deploys ingress-nginx.
🏗️
Terraform IaC
Entire cluster as code — images, containers, volumes, K8s manifests, Helm.
🐘
PostgreSQL
Notes App, Bank Manager (v16), Social Media K8s StatefulSet (v15).
🐬
MySQL 8
Blog Website and Document Intelligence Platform.
🪣
MinIO / S3
S3-compatible object storage for Blog, Document, and Social Media.
🤖
AI / LLM
Google Gemini API for document Q&A. Ollama for local LLM inference.
